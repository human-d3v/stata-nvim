## Syntax

Basic syntax

`svy: tabulate`
[varname](http://www.stata.com/help.cgi?varname)

Full syntax

`svy` \[`vcetype`\] \[`, svy_options`\] `: tabulate`
[varname](http://www.stata.com/help.cgi?varname)
_\[`if`\] \[`in`\]_ \[`, tabulate_options`
`display_items display_options`\]

Syntax to report results

`svy` \[`, display_items display_options`\]
