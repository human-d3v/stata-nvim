## Syntax

------------------------------------------------------------------------

Problem:

. `generate x = 1.1`

. `list`  
(`Stata displays output showing that x is 1.1 in all observations.`)

. `count if x==1.1`  
0

------------------------------------------------------------------------

Solution 1:

. `count if x==float(1.1)`  
100

------------------------------------------------------------------------

Solution 2:

. `generate double x = 1.1`

. `count if x==1.1`  
100

------------------------------------------------------------------------

Solution 3:

. `set type double`

. `generate x = 1.1`

. `count if x==1.1`  
100

------------------------------------------------------------------------
