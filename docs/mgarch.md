## Syntax

`mgarch model eq` \[`eq` ... `eq`\] _\[`if`\]
\[`in`\]_ \[`,` ...\]

|                         |                                  |                                                                                                 |
|-------------------------|----------------------------------|-------------------------------------------------------------------------------------------------|
| Family                  |                                  | `model`                                                                                         |
| Vech                    |                                  |                                                                                                 |
|                         | diagonal vech                    | [<strong>dvech</strong>](http://www.stata.com/help.cgi?mgarch_dvech) |
| Conditional correlation |                                  |                                                                                                 |
|                         | constant conditional correlation | [<strong>ccc</strong>](http://www.stata.com/help.cgi?mgarch_ccc)     |
|                         | dynamic conditional correlation  | [<strong>dcc</strong>](http://www.stata.com/help.cgi?mgarch_dcc)     |
|                         | varying conditional correlation  | [<strong>vcc</strong>](http://www.stata.com/help.cgi?mgarch_vcc)     |
