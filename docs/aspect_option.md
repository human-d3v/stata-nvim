## Syntax

|                                            |                                                   |
|--------------------------------------------|---------------------------------------------------|
| `aspect_option`                            | Description {p2line None}                         |
| `aspectratio:(#` \[`, pos_option`\]`)` | set plot region aspect ratio to `#` {p2line None} |
| `pos_option`                               | Description {p2line None}                         |
| `placement:(compassdirstyle)`          | placement of plot region {p2line None}            |
