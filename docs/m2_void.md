## Syntax

`J(0, 0, .)`

0 `x` 0 real matrix

`J(r, 0, .)`

`r x` 0 real matrix

`J(0,c, .)`

0 `x c` real matrix

`J(0, 0, 1i)`

0 `x` 0 complex matrix

`J(r, 0, 1i)`

`r x` 0 complex matrix

`J(0,c, 1i)`

0 `x c` complex matrix

`J(0, 0, "")`

0 `x` 0 string matrix

`J(r, 0, "")`

`r x` 0 string matrix

`J(0,c, "")`

0 `x c` string matrix

`J(0, 0, NULL)`

0 `x` 0 pointer matrix

`J(r, 0, NULL)`

`r x` 0 pointer matrix

`J(0,c, NULL)`

0 `x c` pointer matrix
