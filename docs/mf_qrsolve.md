## Syntax

`numeric matrix qrsolve(A, B)`

`numeric matrix qrsolve(A, B, rank)`

`numeric matrix qrsolve(A, B, rank, tol)`

`real scalar`<span class="nowrap"> _ `_qrsolve(A, B)`

`real scalar`<span class="nowrap"> _ `_qrsolve(A, B,`
`tol)`

where

`A`: `numeric matrix`

`B`: `numeric matrix`

`rank`: irrelevant; `real scalar` returned

`tol`: `real scalar`
