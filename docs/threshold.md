## Syntax

`threshold`
[depvar](http://www.stata.com/help.cgi?depvar)
\[[indepvars](http://www.stata.com/help.cgi?indepvars)\]
_\[`if`\] \[`in`\]_`, threshvar(varname)`
\[`options`\]

`indepvars` is a list of variables with region-invariant coefficients.
