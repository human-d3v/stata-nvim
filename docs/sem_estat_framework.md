## Syntax

`estat framework` \[`, options`\]

| Options |                | Description                                      |
|---------|----------------|--------------------------------------------------|
|         | `standardized` | report standardized results                      |
|         | `compact`      | display matrices in compact form                 |
|         | `fitted`       | include fitted means, variances, and covariances |
|         | `format(%fmt)` | display format to use                            |
