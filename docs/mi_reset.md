## Syntax

`mi reset`
[varlist](http://www.stata.com/help.cgi?varlist)
\[**=** `exp`\] \[`if`\] \[`, options`\]

| Options |                  | Description                                                                                                                  |
|---------|------------------|------------------------------------------------------------------------------------------------------------------------------|
| Main    |                  |                                                                                                                              |
|         | `m(numlist)` | values of `m` to be reset; default all                                                                                       |
|         | `noupdate`       | see **[<strong>[MI] noupdate option</strong>](http://www.stata.com/help.cgi?mi_noupdate_option)** |
