## Syntax

`mi expand` \[`=`\] `exp`
\[[<strong>if</strong>](http://www.stata.com/help.cgi?if)\]
\[`, options`\]

| Options |                    | Description                                                                                                                  |
|---------|--------------------|------------------------------------------------------------------------------------------------------------------------------|
|         | `generate(newvar)` | create `newvar`; 0=original, 1=expanded                                                                                      |
|         | `noupdate`         | see **[<strong>[MI] noupdate option</strong>](http://www.stata.com/help.cgi?mi_noupdate_option)** |
