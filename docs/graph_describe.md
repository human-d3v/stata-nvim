## Syntax

`graph describe` \[`name`\]

|     |                  |                         |
|-----|------------------|-------------------------|
|     | `name`           | Description             |
|     | `simplename`     | name of graph in memory |
|     | `filename.gph` | name of graph on disk   |
|     | `"filename"` | name of graph on disk   |

If `name` is not specified, the graph currently displayed in the Graph
window is described.
