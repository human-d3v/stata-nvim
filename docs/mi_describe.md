## Syntax

`mi query`

`mi describe` \[`, describe_options`\]

| describe\_options |            | Description                                                                                                                  |
|-------------------|------------|------------------------------------------------------------------------------------------------------------------------------|
|                   | `detail`   | show missing-value counts for `m`=1, `m`=2, ...                                                                              |
|                   | `noupdate` | see **[<strong>[MI] noupdate option</strong>](http://www.stata.com/help.cgi?mi_noupdate_option)** |
