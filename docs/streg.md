`streg`

|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|

|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | `distribution(weibull)`       | Weibull survival distribution                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       | `distribution(lognormal)`     | lognormal survival distribution                                                                                                                  |
